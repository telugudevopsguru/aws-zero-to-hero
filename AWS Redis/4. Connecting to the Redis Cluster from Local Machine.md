### âœ… **Option 2: Use `stunnel` on an Ubuntu Server**

#### ğŸ”¹ **Step 1: Install stunnel**


#### ğŸ”¹ **Step 2: Enable stunnel at Boot**

Edit the default config:

```bash

```

Change:

```

```

To:

```

```

#### ğŸ”¹ **Step 3: Create the Stunnel Configuration**

```bash

```

Example configuration:

```ini

accept = 127.0.0.1:6380
connect = your-redis-endpoint.amazonaws.com:6379
verifyChain = no
```

> ğŸ”¹ `accept`: Local port to connect `redis-cli` to
> ğŸ”¹ `connect`: Remote AWS ElastiCache Redis endpoint
> ğŸ”¹ `verifyChain = no`: Skips cert validation (for testing)

#### ğŸ”¹ **Step 4: Start stunnel**

```bash

```

Or run manually:

```bash

```

#### ğŸ”¹ **Step 5: Connect Using redis-cli (Via Tunnel)**
